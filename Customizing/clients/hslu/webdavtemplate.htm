Anleitung für Windows

Automatische Verbindung mit Windows
    [IF_IEXPLORE]
  1. Starten Sie das Webordner Tool Internet Explorer
    (Sollte ein Fehler auftreten, laden Sie bitte das allgemeine Webordner Tool herunter)
    2. [/IF_IEXPLORE] [IF_NOTIEXPLORE]
  2. Laden Sie das Webordner Tool herunter und starten sie es.
    3. [/IF_NOTIEXPLORE]
  3. Geben Sie die ILIAS-RefId [WEBFOLDER_ID] ein.
  4. Klicken Sie auf "Verbinden".

Manuelle Verbindung mit Windows 7 und Windows Vista

  1. öffnen Sie den Windows-Explorer (z.B. mittels Tastenkombination WindowsLogo + E).
  2. Drücken Sie die Alt-Taste. Das Menü wird eingeblendet.
  3. Wählen Sie "Extras" - "Netzlaufwerk verbinden".
  4. Geben Sie im Feld "Ordner" folgende Adresse ein (Adresse kopieren und einfügen):
    [WEBFOLDER_URI]
  5. Klicken Sie auf "Fertig stellen".
  6. Geben Sie Ihre Anmeldedaten für ILIAS ein und klicken Sie auf "OK".
  7. Bei erfolgreichem Login wird der Webordner geöffnet und steht im Windows-Explorer zur Verfügung.

Die verlangten Anmeldedaten sind identisch mit Ihrem lokalen ILIAS Login und können unter "Persönlicher Schreibtisch - Einstellungen" in ILIAS geändert werden.

Anleitung für Mac OS X

  1. Aktivieren Sie den Finder.
  2. Wählen Sie das Menü 'Gehe zu > Mit Server verbinden...'.
    Dies öffnet das Dialogfenster 'Mit Server verbinden'.
  3. Geben Sie folgende Addresse in das Feld 'Serveradresse' ein:
    [WEBFOLDER_URI]
    und klicken Sie auf 'Verbinden'.
  4. Geben Sie Benutzername und Passwort ein, und klicken Sie auf 'OK'.

Die verlangten Anmeldedaten sind identisch mit Ihrem lokalen ILIAS Login und können unter "Persönlicher Schreibtisch - Einstellungen" in ILIAS geändert werden.

Anleitung für Linux mit Konqueror

  1. Starten Sie den Konqueror Browser.
  2. Geben Sie folgende Adresse in das Adressfeld ein:
    [WEBFOLDER_URI_KONQUEROR]
    und drücken Sie die Eingabetaste.
  3. Geben Sie Benutzernamen und Passwort ein, und klicken Sie auf 'OK'.

Anleitung für Linux mit Nautilus

  1. Starten Sie den Nautilus Browser.
  2. Geben Sie folgende Adresse in das Adressfeld ein:
    [WEBFOLDER_URI_NAUTILUS]
    und drücken Sie die Eingabetaste.
  3. Geben Sie Benutzernamen und Passwort ein, und klicken Sie auf 'OK'.

Die verlangten Anmeldedaten sind identisch mit Ihrem lokalen ILIAS Login und können unter "Persönlicher Schreibtisch - Einstellungen" in ILIAS geändert werden.

Tipps & Support

  • Diese Schritte müssen für einen bestimmten Webordner nur einmalig ausgeführt werden. Sie können später erneut auf die angelegte Verbindung zurückgreifen.
  • Legen Sie Ihre Webordner-Verbindung auf einer übergeordneten Kategorie an. Sie können immer nach "Unten" navigieren, nicht jedoch in höhere Hierarchiestrukturen als Ihr Verbindungspunkt.
  • Falls Ihr Computer den Webordner nicht öffnen kann, finden Sie in der ILIAS Hilfe weitere Hinweise; im Problemfall wenden sich an unseren Support
